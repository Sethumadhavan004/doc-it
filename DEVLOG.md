## Project Overview

`doc-it` is a command-line tool designed to automatically track and document software development sessions. It leverages LangChain and Gemini LLMs to summarize Git commits, identify related past commits, and generate a comprehensive DEVLOG.md file that chronicles the project’s evolution. The core components include a CLI interface, a data layer for managing commit history, and LangChain chains for summarization and pointer detection, all orchestrated within a LangGraph StateGraph for efficient processing.  The project addresses the challenge of maintaining a clear and organized record of development progress, automating the process of creating a detailed DEVLOG.
